# Modify Hosts File (Powershell w/ GUI)

#Description
This is a GUI program which allows us to easily add entries to the windows hosts file. The GUI also provides a button to clear your hosts file completely of all entries.

#Language
Built in Powershell and utilizing some windows APIs, you can execute this program as a Powershell script OR EXE.

#How To Run
You can literally run a Powershell Terminal with Admin Privileges and execute the following command powershell './Modify_Host_File.ps1'. As an alternative take the .exe file and right click, run as ADMINISTRATOR.
